# Advent of Code (Go)

These are my solutions to the advent of code puzzles solved using [Go](https://go.dev/).

## Running the puzzles

Firstly, clone the repository and ensure you have Go installed and that it's available on your `$PATH`. I'm using version `go1.23.3` on linux.

Each puzzle is implemented as a standalone project. 

```bash
cd 2023/day01
```
  
Each puzzle is accompanied by a test suite to validate the solution against the sample inputs and outputs.

You can run the test suite via the test command.
```bash
go test
```

Each puzzle includes the puzzle input. Print the solution by running the application.
```bash
go run main.go
```
